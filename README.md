# Todo_List
## 說明

- 原生 JavaScript
- Vue 3 cdn

### 功能

- 使用者可以點擊按鈕創建待辦事項
- 待辦事項內可以輸入日期、筆記、標註重要性
- 使用者可以依照 icon 進行編輯、完成 task 等動作
- 使用者瀏覽時分為三種模式：全部、呈現進行中 task 、 呈現已完成事項
