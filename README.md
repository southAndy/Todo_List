# Todo_List

## 成果

- [原生 JavaScript](https://github.com/southAndy/Todo_List/Vanilla_JS/index.html)
- [Vue 3 cdn](https://github.com/southAndy/Todo_List/VueJS/index.html)
- [React - hooks (製作中)]()

### 功能說明

1. 使用者可以點擊按鈕創建待辦事項
2. 使用者可以切換為三種狀態的待辦事項：
   a. 全部
   b. 進行中
   c. 已完成
